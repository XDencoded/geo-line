# Geo map

Добавление точек на карте и проложение маршрута между ними.

## Необходимое окружение

Для запуска проекта на ПК должен быть установлен пакетный менеджер. Лучше всего использовать **Bun**.

### Установка Bun

Для установки Bun откройте PowerShell и выполните команду:

```powershell
    powershell -c "irm bun.sh/install.ps1 | iex"
```

### Установка зависимостей

- Перейдите в директорию проекта:

```cmd
    cd /путь/к/вашему/проекту  # например, cd /map-map
```

- Установите библиотеки:

```cmd
    bun i
```
или

```cmd
    bun install
```

### Запуск проекта

После установки зависимостей запустите проект командой:

```cmd
    bun run dev
```
